# React Native Expo Course

## 📚 About
This repository contains learning materials and projects for a React Native with Expo course.

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Expo CLI

### Installation
```bash
npm install -g expo-cli
```

## 🛠️ Tech Stack
- React Native
- Expo
- TypeScript
